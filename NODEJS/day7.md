# Git and GitHub

- tracks changes, branch, repo etc
- git init (initialises git repo)
- git status (shows the current dir status)
- git add (to stage the files or track the files)
- git commit -m "message" (saves the staged changes permanently to the local Git repository with a descriptive message explaining what was changed)
- git diff file.txt (shows the file changes +green if added -red if removed or edited)
- git log (shows all the commits we've made)
- MASTER vs MAIN branch
- git reset file.txt (used to unstage files)
- git restore file.txt (unstages file and deletes everything upto the last commit)
